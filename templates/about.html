About This Project

The House Price Predictor is a machine learning web application designed to estimate house prices based on input features such as area, number of bedrooms, bathrooms, amenities, and location preferences.

Built using Python, Flask, and a trained regression model, this app provides quick and easy predictions for real estate evaluations. Whether you're a home buyer, seller, or real estate agent, this tool can help you understand market value better.

The interface is styled with HTML5, CSS3, and Font Awesome icons for a clean and modern experience.
